How to create Facebook Group

Our today's post is how we can create a Facebook group so let's see how to create a group


1. First Open Any Facebook App


2. Click Menu Button


3. Click Group Option


4. Click On + Icon


5. Click Create Group Option


6. Now here you will give the name of your group and after that you will give the group whether you want to make it private or public Click on Create Group and it will be done
